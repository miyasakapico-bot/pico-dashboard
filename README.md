# 🍎 Pico Dashboard Ver1

毎日開いて、生活・勉強・趣味の小さな記録をまとめる、自分専用のダッシュボードです。

Ver1は **HTML・CSS・JavaScriptだけ** で作られています。サーバーや会員登録は不要です。入力した内容は、使っているブラウザの `localStorage`（ブラウザ内の保存場所）へ保存されます。

## 起動方法

1. このフォルダーにある `index.html` をダブルクリックします。
2. Chrome、Edge、SafariなどのブラウザでPico Dashboardが開きます。
3. チェックや一言、気分、画像を登録して使います。

特別なインストールやコマンド入力は必要ありません。

> 大切：ブラウザの閲覧データやサイトデータを消すと、Pico Dashboardの記録も消える場合があります。Ver1では同じ端末・同じブラウザで使ってください。

## Ver1でできること

- 今日の日付の表示
- 7項目の達成チェックと達成率バー
- 今日の一言の保存
- 6種類から今日の気分を選択
- 今日のイラストを1枚保存・拡大表示
- ぽふりん画像を1枚保存・拡大表示
- 直近7日間の履歴表示
- 3種類の勉強達成日数と連続記録の集計
- スマートフォンに合わせた表示

## データ保存について

記録はブラウザの `localStorage` に保存されます。画像は保存しやすい大きさへ自動的に縮小されますが、ブラウザの保存容量には上限があります。容量がいっぱいになった場合は、古い画像を削除してください。

Ver1はバックアップ機能がまだありません。大切な元画像は、パソコンやスマートフォンにも残してください。

## GitHubへ登録する方法（初心者向け）

まずはGitHubの無料アカウントを作ります。その後、いちばん分かりやすい **GitHub Desktop** を使う方法がおすすめです。

1. [GitHub](https://github.com/) でアカウントを作ります。
2. [GitHub Desktop](https://desktop.github.com/) をインストールして、GitHubアカウントでサインインします。
3. GitHub Desktopで **File → Add Local Repository** を選びます。
4. このPico Dashboardのフォルダーを選びます。
5. リポジトリではないと表示されたら **create a repository** を押します。
6. Nameを `pico-dashboard` にし、**Create Repository** を押します。
7. 左下のSummaryへ `Pico Dashboard Ver1` と入力し、**Commit to main** を押します。
8. 上部の **Publish repository** を押します。
9. 公開してよければ **Keep this code private** のチェックを外します。自分だけにしたい場合はチェックを付けたままにします。
10. もう一度 **Publish Repository** を押せば登録完了です。

将来、GitHub Pagesを使えば、インターネット上でPico Dashboardを開けるようにもできます。ただしlocalStorageの記録は端末・ブラウザごとに別です。
